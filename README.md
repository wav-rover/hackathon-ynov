# Hackathon Ynov — Vet Locator

Hackathon project monorepo: React frontend + backend API.

## Structure

```
├── front/   → React app (Vite + shadcn/ui)
└── back/    → Backend API
```

## Quick start

**Frontend**

```bash
cd front
npm install
npm run dev
```

**Backend**

```bash
cd back
# configure based on chosen stack
```
